# ⌨️ Горячие клавиши NAME AGENCY - NeuroDesk IDE

## Основные команды

### Навигация и поиск
- `Ctrl+Shift+P` - **Command Palette** (командная палитра)
- `Ctrl+P` - Быстрый поиск файлов (в разработке)
- `Ctrl+,` - Открыть настройки (в разработке)

### Панели
- `Ctrl+`` - **Toggle Terminal** (показать/скрыть терминал)
- `Ctrl+Shift+G` - **Toggle Git Panel** (показать/скрыть Git панель)
- `Ctrl+Shift+E` - Показать Explorer (в разработке)
- `Ctrl+B` - Toggle Sidebar (в разработке)

### Редактор
- `Ctrl+S` - **Сохранить файл**
- `Ctrl+W` - Закрыть текущий файл (в разработке)
- `Ctrl+Tab` - Переключение между файлами (в разработке)
- `Ctrl+F` - Поиск в файле (Monaco Editor встроенный)
- `Ctrl+H` - Замена в файле (Monaco Editor встроенный)
- `Ctrl+/` - Закомментировать строку (Monaco Editor встроенный)

### AI Ассистент
- `Ctrl+Enter` - **Отправить сообщение AI** (в чате)
- `Ctrl+Shift+A` - Открыть AI панель (в разработке)
- Быстрые действия в AI панели:
  - **Explain** - Объяснить код
  - **Refactor** - Рефакторинг
  - **Document** - Добавить документацию
  - **Optimize** - Оптимизировать код

### Git
- `Ctrl+Shift+G` - **Открыть Git панель**
- `Ctrl+Enter` - **Commit** (в Git панели)
- AI генерация commit message - кнопка "AI" в Git панели

### Терминал
- `Ctrl+`` - **Toggle Terminal**
- `Ctrl+Shift+`` - Новый терминал (в разработке)
- `clear` - Очистить терминал
- `help` - Показать доступные команды
- `echo <text>` - Вывести текст

### Command Palette (Ctrl+Shift+P)
- `↑` / `↓` - Навигация по командам
- `Enter` - Выполнить команду
- `Esc` - Закрыть палитру

### Окно приложения
- `Alt+F4` - Закрыть приложение
- Кнопки в Title Bar:
  - Minimize (минимизировать)
  - Maximize (развернуть)
  - Close (закрыть)

## Monaco Editor (встроенные)

### Редактирование
- `Ctrl+X` - Вырезать строку
- `Ctrl+C` - Копировать строку
- `Ctrl+V` - Вставить
- `Ctrl+Z` - Отменить
- `Ctrl+Y` - Повторить
- `Ctrl+D` - Выделить следующее вхождение
- `Alt+↑/↓` - Переместить строку вверх/вниз
- `Shift+Alt+↑/↓` - Копировать строку вверх/вниз
- `Ctrl+Shift+K` - Удалить строку

### Навигация
- `Ctrl+G` - Перейти к строке
- `Ctrl+Home` - В начало файла
- `Ctrl+End` - В конец файла
- `Ctrl+←/→` - Перемещение по словам
- `Home` - В начало строки
- `End` - В конец строки

### Выделение
- `Shift+←/→/↑/↓` - Выделение
- `Ctrl+Shift+←/→` - Выделение по словам
- `Ctrl+A` - Выделить всё
- `Alt+Click` - Множественный курсор
- `Ctrl+Alt+↑/↓` - Добавить курсор выше/ниже

### Поиск и замена
- `Ctrl+F` - Поиск
- `Ctrl+H` - Замена
- `F3` - Следующее совпадение
- `Shift+F3` - Предыдущее совпадение
- `Ctrl+Shift+F` - Поиск по всем файлам (в разработке)

### Форматирование
- `Shift+Alt+F` - Форматировать документ
- `Ctrl+K Ctrl+F` - Форматировать выделение
- `Ctrl+]` - Увеличить отступ
- `Ctrl+[` - Уменьшить отступ

### IntelliSense
- `Ctrl+Space` - Показать подсказки
- `Ctrl+Shift+Space` - Показать параметры функции
- `F12` - Перейти к определению
- `Alt+F12` - Peek определение
- `Shift+F12` - Найти все ссылки

## Настройка горячих клавиш

Горячие клавиши можно настроить в файле:
`src/App.tsx` - функция `handleKeyDown`

Пример добавления новой горячей клавиши:

```typescript
if (e.ctrlKey && e.key === 'k') {
  e.preventDefault()
  // Ваше действие
}
```

## Советы

1. **Command Palette** (`Ctrl+Shift+P`) - самый быстрый способ найти любую команду
2. **Monaco Editor** имеет все горячие клавиши VS Code
3. **Терминал** поддерживает базовые команды (clear, help, echo)
4. **AI панель** - используйте быстрые кнопки вместо ввода промптов вручную
5. **Git панель** - кнопка "AI" автоматически создаст commit message
